## Installation

1. Clone the repository in your local folder.
```
git clone https://github.com/Juakata/Portfolio-V2.git
```
2. Before start run
```
bundle install
```
```
yarn install --check-files
```
3. To open the project run
```
rails server
```
```
go to http://localhost:3000/
```
